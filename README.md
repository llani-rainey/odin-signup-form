# ⚡ Odin Sign-up Form

This project is a sign-up form for an imaginary service, designed to practice layout scaffolding, form validation, and advanced CSS concepts. It is a part of **The Odin Project's** Intermediate HTML and CSS curriculum.

## 🚀 Features

* **Custom Sidebar:** Includes a large background image from Unsplash and an Odin logo.
* **Logo Banner:** A semi-transparent background div behind the "ODIN" logo to enhance readability against the background image.
* **Themed Styling:** The 'Create Account' button uses the specific moss green color (#596D48) as requested in the design brief.
* **Form Validation:**
    * **Invalid State:** Password inputs feature a red border when the content is invalid, utilizing the `:invalid` pseudo-class.
    * **Focus State:** Selected inputs feature a blue border and a subtle box-shadow using the `:focus` pseudo-class.
* **Typography:** Integration of the **Norse Bold** font for the logo section.



## 🛠️ Built With

* **HTML5:** Semantic structure including a sidebar and content area.
* **CSS3:** Flexbox for layout, custom fonts, and pseudo-class validation.
* **Git/GitHub:** Version control and terminal-based workflow.

## 📝 Project Requirements Met

* **Layout:** Scaffolding the page structure section by section.
* **Assets:** Use of external fonts and high-resolution background imagery.
* **Input Styling:** Default light borders (#E5E7EB) with dynamic variations for focus and error states.
* **Note:** This version does not include JavaScript for password matching or mobile responsiveness, as per project instructions.
